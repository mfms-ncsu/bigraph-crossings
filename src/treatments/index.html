Source code for treatments

  • BiGraph.cxx - bipartite graph class for crossing number problem
  • BiGraph.hxx - bipartite graph class for crossing number problem
  • CmdLine.cpp - implementation of command-line parser
  • CmdLine.h - header for a class that parses the command line
  • IntSet.cxx - set of integers for use with graph adjacency lists
  • IntSet.hxx - set of integers for representing adjacency lists in graphs
  • Queue.hxx - queue of integers, implemented as a linked list
  • Sums.cxx - data structure for efficient retrieval and update of sums of contiguous integers in an indexed list
  • Sums.hxx - data structure for efficient retrieval and update of sums of contiguous integers in an indexed list
  • adaptive_insert.cxx - implementation of (one iteration of) the adaptive insertion heuristic
  • adaptive_insert.h - header for adaptive insertion heuristic
  • average.cxx - implements median/barycenter heuristics
  • average.h - header for median/barycenter heuristics
  • bfs_heuristic.cxx - implementation of a bfs-based heuristic (gbfs)
  • bfs_heuristic.h - header for bfs-based heuristic (gbfs)
  • cn_eval.cxx - reports number of crossings for a graph, given an ord file
  • defs.h - definitions common to all edge crossing heuristic source files
  • dot.cxx - functions that read "dot" format for graphs
  • dot.h - module for reading "dot" files
  • fstats.cxx - compile simple statistics for experimental data
  • gstats.cxx - reports various statitics about a bigraph in dot format
  • heuristic.cxx - implements heuristics for edge crossings
  • ord.cxx - implementation of utility functions that read and write .ord files (node ordering on layers of a graph)
  • ord.h - header for utility functions that read and write .ord files (node ordering on layers of a graph)
  • plain_bfs.cxx - implementation of a breadth-first search
  • plain_bfs.h - header for plain bfs ordering
  • tree_lower_bound.cxx - implementation of lower bound for trees
  • tree_lower_bound.h - header for tree lower bound calculation
  • Makefile for programs related to edge crossing minimization version 1.08 (the version used for the JEA paper)
    Uses GNU-style syntax (also works on Sun OS and Solaris)
